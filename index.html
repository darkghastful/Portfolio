Interactive University Data

This is an interactive shiny page that allows users to view statistics for universities across the US. Data was sourced from The National Center for Education Statistics (NCES) Integrated Postsecondary Education Data System (IPEDS). The code for this project was written in R, and functions used to process the data and generate the plots can be found in packages on my page: bqutils and bitblot.
